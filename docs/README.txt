リリース番号は 3 桁で構成されます。それぞれ、上からメジャーリリース番
号、マイナーリリース番号、アップデート番号、となります。

メジャーリリースおよびマイナーリリース番号はライブラリが変更された場合、
あるいはアーキテクチャが変わった場合に加算され、アップデート番号はバー
ジョンに関係なくこれまでに何回アップデートされたかをあらわします。

全てのサポート、及び問い合わせ、パッチ提供はこれらのリリース番号をキー
として行います。

					リリース		   1.1
					==============================

					アップデート		1.1.32
					==============================



LIBC はライブラリ、オブジェクト、インクルードファイルからそのソースコー
ドにいたるまで、そのすべてが PDS として公開されています。誰でも自由に
LIBC を使えるし、コピーすることも、改変することも、また自分のソフトウェ
アに組み込むこともできます。

LIBC に関するすべての著作権は放棄されています。また放棄できない部分に
ついても、LIBC が意味的に PDS として成立するよう、著作権を主張しないこ
とが宣言されています。

また同時に、LIBC は無保証です。LIBC を用いて作成したプログラムがライブ
ラリの不具合によって意図しない動作をし、それによって何らかの被害が生じ
たとしても、作成者である Project LIBC Group は一切責任をおいません。
